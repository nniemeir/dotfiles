# Keybindings
## CLI Programs
| Keybinding | Action                |
|------------|-----------------------|
| Win+h      | Control Panel         |
| Win+e      | Ranger                |
| Win+Return | Terminal              |

## GUI Programs
| Keybinding  | Action                |
|-------------|-----------------------|
| Win+Shift+F | Firefox               |
| Win+Shift+V | Pavucontrol           |
| Win+Shift+E | Thunar                |

## Compositor Controls
| Keybinding       | Action                               |
|------------------|--------------------------------------|
| Win+Ctrl+Shift+L | Swaylock                             |
| Win+Ctrl+w       | Reload Sway                          |
| Win+Ctrl+q       | Exit Sway                            |
| Win+Ctrl+x       | Kill Focused Window                  |
| Win+Ctrl+h       | Focus Left                           |
| Win+Ctrl+j       | Focus Up                             |
| Win+Ctrl+k       | Focus Down                           |
| Win+Ctrl+l       | Focus Right                          |
| Win+Ctrl+f       | Toggle Focused Window Fullscreen     |
| Win+Ctrl+Space   | Toggle Focused Window Tiled/Floating |
| Win+Ctrl+Left    | Workspace Left                       |
| Win+Ctrl+Right   | Workspace Right                      |
| Win+[1-6]        | Switch To Workspace X                |
| Win+Ctrl+[1-6]   | Move Focused Window To Workspace X   |
