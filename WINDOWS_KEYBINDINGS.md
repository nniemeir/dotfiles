# Windows Keybindings
These are added through AutoHotkey in addition to the existing Windows keybindings.

## CLI Programs
| Keybinding      | Action                   |
|-----------------|--------------------------|
| Win+Shift+E     | Lf                       |
| Win+Enter       | Windows Terminal         |
| Win+Shift+Enter | WSL Debian Terminal      |

## GUI Programs
| Keybinding  | Action                |
|-------------|-----------------------|
| Win+Shift+F | Firefox               |
| Win+Shift+V | Volume Mixer          |
| Win+Shift+E | Thunar                |

## Window Manager Controls
| Keybinding       | Action                               |
|------------------|--------------------------------------|
| Win+n            | Minimize Focused Window              |
| Win+m            | Maximize Focused Window              |
| Win+f            | Toggle Focused Window Fullscreen     |
| Win+t            | Toggle Focused Window Always On Top  |

## Additional Actions
| Keybinding       | Action                               |
|------------------|--------------------------------------|
| Win+Del          | Empty Recycle Bin                    |
